geolatte-graph
==============

A library for graph algorithms and data structures.

geolatte-graph offers the following:

Data Structures
=====================
* Pairing Heap
* Red-Black Tree
* Spatial Index (Grid)
* Grid-indexed graphs

Algorithms
==========
* Shortest Path
* Dijkstra
* Extensible A*
* Distance-limited Breath-first
